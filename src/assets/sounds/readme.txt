Sound File Placeholders

This directory should contain the following sound files in MP3 format:

1. rain.mp3 - Gentle rain background sound
2. ocean.mp3 - Ocean waves sound
3. forest.mp3 - Forest ambiance sound
4. birds.mp3 - Bird chirping sounds
5. chime.mp3 - Interval notification chime
6. end-chime.mp3 - End of meditation session chime

For the application to function properly, please add these sound files or replace them with your own custom sounds.

You can find free sound files from various sources such as:
- Freesound.org
- Zapsplat.com
- SoundBible.com

Make sure all sounds are properly licensed for your use case.
